BOIDS!!!! is a control the swarm style game that allows the player to control different aspects of flocking agents.
Bounding box size controls the distance at which agents are redirected back towards the origin.
Bounding box solidity controls the speed at which agents are redirected back towards the origin.
Move towards center, repel each other, and match velocities control coefficients of v1, v2, and v3 respectively.
Add boid adds an agent at the origin.
Remove boid removes an agent, never letting the number of agents go below 2.
